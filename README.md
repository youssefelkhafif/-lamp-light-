# Lamp Light Project 💡

A simple interactive project to control and simulate lamp/light behavior. This project demonstrates how to turn a lamp **on/off**, adjust brightness, and explore light simulation features.  

## Features

- **Turn lamp on/off**  
- **Adjust brightness levels** (if supported)  
- **Interactive interface** to control the lamp (GUI or command-line)  
- **Light simulation** to visualize lamp behavior  
- (Optional) Timers or color effects
